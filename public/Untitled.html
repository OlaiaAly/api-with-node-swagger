User Management API

     

        This API provides endpoints for managing users, including fetching,         updating, and deleting user data. It's built using Fastify and Prisma,         with Zod for schema validation.      

     

Key Features

     
           
  •           Comprehensive user management: Fetch, update, and delete user data.        
    •        
  • Built with modern technologies: Fastify, Prisma, and Zod.
    •        
  • Schema validation: Ensures data integrity with Zod.
    •        
  • Easy to use: Clear API documentation with Swagger UI.
